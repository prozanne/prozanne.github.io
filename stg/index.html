TOAST is very useful tool for TV Web Application developer.
This is Samsung's open source project that was created to minimize overhead in TV web application development. This will enable developers to generate a multiple-platform solution for all of their TV web applications. We hope you can develop a multiple platform solution for ALL your TV web applications.

About TOAST

Multiple Platform

Multiple platform development solution for TV web application. Write once, generate multiple TV app packages.

Hosted On Github

cordova-plugin-toast

Cordova plugin that provides TV Specific functionalities.
Currently we support following features: toast.Media, toast.MediaPlugin, toast.drminfo, toast.application, toast.inputdevice, toast.tvaudiocontrol, toast.tvwindow, toast.tvchannel

grunt-cordova-sectv

Grunt task for prepare and build the cordova project with sectv-orsay and sectv-tizen platforms.

cordova-sectv-tizen

cordova-sectv-tizen is an TV application library that allows for Cordova-based projects to be built for Samsung Tizen TV Platform.

cordova-sectv-orsay

cordova-sectv-orsay is an TV application library that allows for Cordova-based projects to be built for the Legacy Samsung Smart TV (A.K.A Orsay) Platform.

cordova-tv-webos

cordova-tv-webos is an TV application library that allows for Cordova-based projects to be built for the WebOS TV Platform.
